Generic Models

Motivation

Going through simpler models and following the steps of

  1. envisioning
  2. quantifying
  3. constructing
  4. testing (via Graphs and Tables)
is very tedious. To construct models for the complicated scenarios, these steps become time consuming and mathematically overwhelming. So, to further ones' learning they need to break the complicated processes to elementary models. These elementary models are called template models. These templates interact through nodes to create more complicated dynamic systems. Lets go through them now.

Introduction

Dynamic models show us how value is changing from one quantity to another. This process is called transfer. To give a general names to these quantities:

  1. Source: the quantity which decreases.
  2. Destination: quantity which increases
  3. Changing: the factor by which the quantity is changing.
If the values are linearly, that is if the source is decreasing by at a pace and destination is increasing by a constant factor of source then the transfer process is called linear. Similarly if the transfer is proportional to the source or destination then it is called exponential. In this exponential process if the system is trying to achieve a set point, then the process is called homeostasis. To summarize, the whole transfer process is always between two quantities, and the changing factor defines what kind of process it is.

Now you will solve some Dragoon Problems where you will see such template models working.

Instructions

Every problem has some idea attached to it, which will be in the form of lessons learned. Once you think you have completed the problem then you should check the graph. It gives a feedback whether the model you have constructed matches the authored model or not. If it does, then as soon as you will close the graph you will see that a Lessons Learned window will pop up. This window will try to help you understand the process that you were creating as per the templates explained above. You might have realized uptill now that

  • Dodos population was exponential decay process
  • Rod's bookworm problem was a linear growth process
  • Rabbits population was exponential growth process
. Did you realize what was the difference in decay and growth processes.
